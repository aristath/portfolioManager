# Legacy Python Code - Migration Reference Only

This directory contains the legacy Python trading application that is being migrated to Go.

**Status:** Non-functional - For reference only during migration

## Contents

- `app/` - Legacy Python application code
- `tests/` - Legacy Python tests
- Python config files (pyproject.toml, pytest.ini, requirements.txt, etc.)

## Purpose

This code is kept around temporarily as a reference while completing the Python-to-Go migration. Once the migration is complete, this entire directory will be deleted.

## Migration Status

See the main MIGRATION-REPORT.md in the project root for detailed migration status and remaining work.

## DO NOT USE

This code is **not functional** and should not be executed or relied upon. All active development and operations use the Go codebase in `/trader`.
